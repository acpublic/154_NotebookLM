# NotebookLM
- ユーザーが指定したソースの内容だけをもとに、質問応答や要約、アイデア出しなどを行う、GoogleのAIノートツール
## 使い方
### NotebookLM起動
- NotebookLMのサイトにアクセスし、Googleアカウントでログイン
- 新しいノートブックを作成
### ソースの追加
- Googleドライブ
- PDF
- テキスト
- ウェブサイトのURL
- YouTube:字幕あり
### チャット
- 対話
### Studio
- 音声解説
- 動画解説
- マインドマップ
- レポート
- フラッシュカード
- クイズ
- インフォグラフィックス
- スライド資料
- Data Table
